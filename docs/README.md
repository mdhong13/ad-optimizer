# docs/

광고 최적화 프로젝트 참조 문서 보관 폴더.
코드가 아닌, 프로젝트에서 참조하는 외부 자료들을 보관합니다.

## 구조

```
docs/
├── references/     참조 문서 (API 가이드, 광고 정책, 크립토 시장 자료 등)
└── guides/         내부 가이드 (운영 매뉴얼, 설정 가이드 등)
```

## 추천 보관 문서
- Meta Ads 정책 가이드 (크립토 광고 제한 사항)
- Google Ads 크립토 광고 심사 기준
- OneMessage 앱 소개서 / 피치 덱
- 타겟 페르소나 정의서
- 경쟁사 분석 자료
