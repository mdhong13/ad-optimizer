# Daily Report (일간 광고 성과 리포트)

## 역할
광고 자동화 시스템의 **하루 단위 운영 보고서**를 생성한다. 전날 집행된 모든 캠페인의 성과를 요약하고, 이상 징후를 잡아내고, 오늘 취할 액션을 제안한다.

## 데이터 소스
- DB `campaigns` 컬렉션: 캠페인 메타데이터
- DB `performance` 컬렉션: 시간별 성과 지표
- DB `decisions` 컬렉션: AI 결정 이력
- Meta/Google/X/Reddit API: 최신 실시간 지표 (전일 최종 값 보정)

## 리포트 구조

### 1. 헤드라인 (한 줄 요약)
```
📊 2026-04-19: 지출 ₩32,400 (80% 소진) | 전환 14건 | CPA ₩2,314 (-18% MoM)
```

### 2. 플랫폼별 성과 테이블
| 플랫폼 | 지출 | 노출 | 클릭 | CTR | CPA | ROAS | vs 어제 |
|--------|------|------|------|-----|-----|------|---------|
| Meta | ₩18k | 24k | 380 | 1.58% | ₩2,250 | 2.4x | 📈 +8% |
| Google | ₩10k | 8k | 120 | 1.50% | ₩2,500 | 2.1x | 📉 -5% |

### 3. 캠페인 Top/Bottom 3
- 🏆 승자 3: 왜 좋았나 + 유지/확대 제안
- ⚠️ 패자 3: 왜 나빴나 + 중단/수정 제안

### 4. 이상치 알림
- CTR/CPA 2σ 벗어난 건
- 예산 90%+ 소진된 캠페인
- 에러로 실패한 크리에이티브
- 좀비(PAUSED지만 과금 걸린 건)

### 5. 오늘 액션 아이템
우선순위 순으로 3~5개:
1. [즉시] 캠페인 X 중단 (CPA 4배)
2. [오늘] 캠페인 Y 예산 +20%
3. [이번주] 신규 A/B 실험 X

### 6. 실험 진행 현황
- 돌고 있는 실험: [이름, 진행도, 샘플 도달률]
- 오늘 종료 예정: [이름 → 결론 도출 필요]

## 작업 지침

1. **날짜 범위 확인**: "어제"는 사용자 타임존(Asia/Seoul) 기준 00:00~23:59. UTC 혼동 주의.

2. **데이터 보정**: 플랫폼마다 전날 최종 지표 확정까지 6~24시간 딜레이. 리포트에 **"잠정"** 표시.

3. **비교 기준**: vs 어제 / vs 7일 평균 / vs 전월 동일 요일 중 가장 의미 있는 것 선택. 단순 전일 비교는 요일 효과로 왜곡.

4. **이상치 정의**:
   - CTR: 베이스라인 ±2σ
   - CPA: 베이스라인 ±50% (변동 크므로 넓게)
   - 지출: 일 예산 캡 근접(>85%) 또는 전일 대비 2배
   - 에러: API 호출 실패율 >5%

5. **액션은 구체적으로**: "모니터링 강화" 같은 모호한 지시 금지. "캠페인 X를 PAUSED로 전환" 수준의 실행 가능한 명령.

6. **좀비 탐지**: 지난 7일 지출 0원인데 ACTIVE인 캠페인, 30일 이상 실행된 저성과 캠페인 → 플래그.

7. **포맷**: 한 화면에 읽을 수 있게 축약. 상세는 별도 링크. Gmail/Slack 전송 고려.

## OneMessage 프로젝트 맥락
- 타임존: Asia/Seoul (UTC+9)
- 통화: KRW
- 일 예산 캡: ₩40,000
- 사이클: 8시간마다 최적화
- 이상치 민감도: 소규모 예산이라 노이즈 큼 → σ 기준 너무 타이트하면 False Positive 많음
