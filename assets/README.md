# assets/

광고 이미지/영상 소재 보관 폴더.

## 구조

```
assets/
├── references/         참조 이미지 (경쟁사 광고, 벤치마크, 디자인 레퍼런스)
└── generated/          생성된 광고 소재
    ├── meta/           Meta (FB/Instagram) 규격
    ├── google/         Google Ads 규격
    ├── twitter/        Twitter/X 규격
    └── blog/           블로그 썸네일 / 본문 이미지
```

## 플랫폼별 이미지 규격

| 플랫폼 | 유형 | 권장 크기 |
|--------|------|----------|
| Meta Feed | 단일 이미지 | 1080x1080 또는 1200x628 |
| Meta Story | 세로 | 1080x1920 |
| Google 반응형 | 가로 | 1200x628 |
| Google 반응형 | 정사각 | 1200x1200 |
| Twitter | 단일 이미지 | 1200x675 |
| Blog 썸네일 | OG 이미지 | 1200x630 |
